# 0.获得的数据是怎么组成的？

## 表达矩阵

### example

```         
           sample1  sample2  sample3
GeneA        5.2      4.9      6.1
GeneB        0.3      0.1      0.2
GeneC       10.8     11.2      9.9
```

-   行：被测的对象（基因 / 探针）
-   列：一次完整的观测（一个人 / 一块组织）
-   数值：仪器给出的信号强度（RNA 多不多）

## 样本信息（条件/状态）

### example

```         
        disease   age   sex
sample1   AD      72    F
sample2   AD      68    M
sample3 Control   70    F
```

## 基因注释（把基因翻译成人话）

### example

```         
        GeneSymbol   EntrezID    Description
GeneA      TP53        7157    tumor suppressor
GeneB      IL6         3569    cytokine
```

# 1.总体的流程

## 0.数据来源

| 数据源   | 在 R 里的“容器”      |
|----------|----------------------|
| GEO      | ExpressionSet        |
| TCGA     | SummarizedExperiment |
| 自己实验 | matrix / data.frame  |

## 1.统一处理

| 数据源 | 表达矩阵           | 样本信息             |
|--------|--------------------|----------------------|
| TCGA   | `assay(tcga_data)` | `colData(tcga_data)` |
| GEO    | `exprs(gse)`       | `pData(gse)`         |

## 2.质量控制（QC）

### 基因层面

``` r
# 例：去掉几乎不表达的基因
expr <- expr[rowSums(expr > 1) >= 10, ]
```

### 样本层面

``` r
boxplot(log2(expr + 1))   # 例：样本分布是否离谱
```

## 3.标准化

### GEO

``` r
expr <- normalizeBetweenArrays(expr)
expr <- log2(expr + 1)
```

### TCGA

``` r
dds <- DESeqDataSetFromMatrix( # 把数据装进一个统计模型容器
  countData = expr,
  colData = pheno,
  design = ~ group_list
)
dds <- DESeq(dds) # 标准化
expr <- assay(vst(dds)) # 把 count 数据变成“看起来像芯片数据”的形式
```

## 4.分组

``` r
group_list <- ifelse(
  pheno$sample_type == "Solid Tissue Normal",
  "normal",
  "tumor"
)
group_list <- factor(group_list, levels = c("normal", "tumor"))
```

## 5.差异分析

``` r
design <- model.matrix(~ group_list)

deg <- lmFit(expr, design) |>
  eBayes() |>
  topTable(coef = 2, number = Inf)
```

### deg

| 列名      | 含义            |
|-----------|-----------------|
| logFC     | 差异方向和大小  |
| P.Value   | 显著性          |
| adj.P.Val | 多重校正后 P 值 |

## 6.标记上下调

``` r
deg$change <- "stable"
deg$change[deg$logFC > 1 & deg$P.Value < 0.05] <- "up"
deg$change[deg$logFC < -1 & deg$P.Value < 0.05] <- "down"
```

## 7.最后成果

-   deg（差异基因表）
-   上调 / 下调基因列表
-   送去画图

# 3.deg详解

## 前置条件

``` r
expr        # 表达矩阵：行=基因，列=样本
pheno       # 样本信息
group_list  # 样本分组（normal / tumor）
```

## 1.构建设计矩阵（告诉R比较对象）

``` r
design <- model.matrix(~ group_list)
```

### 输出：

| 样本    | normal 组？ | tumor 组？ |
|---------|-------------|------------|
| sample1 | 是          | 否         |
| sample3 | 否          | 是         |

## 对每个基因做数学比较

### 拟合线性模型

``` r
# 对 每一个基因用 design 里的分组信息拟合一个模型
# 比较两组差异
fit <- lmFit(expr, design)
```

### 经典贝叶斯

``` r
# 让所有基因“互相参考”来稳定统计结果
fit2 <- eBayes(fit)
```

### 生成deg

``` r
# coef = 2 = 比较 tumor vs normal
deg <- topTable(fit2,coef = 2, number = Inf)
```

# 4.关于画图

## 0.概览

``` md
所有图 = 从一个表中
1.选几列
2.指定“谁当 x，谁当 y，谁当颜色/形状”
```

### 核心架构

``` r
# 可以理解为指定哪列画横轴，哪列画纵轴，哪列用颜色区分
ggplot(data, aes(x = ?, y = ?, color = ?)) + 
geom_xxx() # 画什么图
```

## 1.火山图

### 回答：哪些基因变得多又显著？

### 数据来源：deg表

### 设计逻辑

| 轴    | 含义                |
|-------|---------------------|
| x     | logFC（变化大小）   |
| y     | -log10(P)（显著性） |
| color | 上调 / 下调         |

### 示例代码
```
library(ggplot2)

ggplot(deg, aes(x = logFC, y = -log10(P.Value), color = change)) +
  geom_point(alpha = 0.6) +
  theme_minimal()
```
